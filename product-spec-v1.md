# Product: Scanly
**Tagline:** Smarter market scanning

## One-liner
A fast crypto market scanner that explains momentum and risk using data and AI.

## Target User
Retail crypto traders using centralized exchanges.

## Core Pages
- Market Scanner
- Token Detail
- Watchlist
- Login
- About

## MVP Features
- Market scanning (price, volume)
- Token charts
- RSI + EMA
- AI summary
- Watchlist

## Explicitly Excluded
- On-chain analytics
- Signals
- Portfolio tracking
